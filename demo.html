author：qipbbn**gmail.com(@maxbbn)
disc：基于HTML5 Canvas 的图表引擎
git：maxbbn/cchart

功能点

  • 目前支持条形图，折线图, 饼图， 其他图表正在调研中
  • 自动颜色配置，支持颜色自定义
  • 数字显示，格式化
  • 自定义动画效果
  • Y坐标自适应
  • 多种JSON数据定义格式
  • 通过 flashCanvas 添加对IE的支持

Demo - 条形图（Bar Chart）

{
    type : "bar",
    elements : [
        {
            name : "接待人数",
            datas : [8,29,22,31,35,40],
            label : "{name} : {data}人"
        },
        {
            name : "回复人数",
            datas : [18,19,23,38,45,30],
            label : "{name} : {data}人"
        }
    ],
    axis : {
        x : {
            name : "时间",
            labels : ["2011-1-10","2011-1-11","2011-1-12","2011-1-13","2011-1-14","2011-1-15"]
        },
        y : {
            name : "人数"
        }
    },
    config : {
        left:30
    }
}

null
